## 🔧 КАК ЭТО РАБОТАЕТ

### Структура данных в Tracker API

```python
changelog = issue.changelog
# Возвращает список Records с информацией о изменениях

for record in changelog:
    record.updatedAt        # Дата/время события
    record.updatedBy        # Кто произвел изменение
    record.type             # Тип события (IssueUpdated, IssueCreated и т.п.)
    record.fields[]         # Список измененных полей:
                            # [{'field': Reference, 'from': old, 'to': new}, ...]
```

### Парсинг переходов в RC

```python
for record in changelog:
    for field_change in record.fields:
        field = field_change['field']
        from_status = field_change['from']
        to_status = field_change['to']
        
        # Ищем изменение статуса на RC
        if 'status' in str(field).lower() and 'RC' in str(to_status):
            rc_enter_date = record.updatedAt  # ← точная дата!
```
